新手三步

三步启动

  1. 下载并解压对应系统的内测包（Windows 或 macOS）。
  2. 启动桌面助手，在 API Settings 填入你的模型 API 信息。
  3. 点击一键启动，导入历史资料和邮件线程开始使用。
Windows 安装

Windows 安装

  1. 先解压到本地目录，不要在压缩包内直接运行。
  2. 双击 Launch_InVault_Windows.bat。
  3. 若出现安全提示，选择“更多信息 -> 仍要运行”。
macOS 安装

macOS 安装

  1. 首次被拦截时，打开“系统设置 -> 隐私与安全性”。
  2. 在底部找到应用拦截提示，点击“仍要打开”。
  3. 若仍提示损坏，在终端执行：xattr -dr com.apple.quarantine /Applications/InVault.app。
